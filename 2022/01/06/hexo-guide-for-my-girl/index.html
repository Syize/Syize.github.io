布置到 Github 上

Github 上的设置

首先打开你的 Github 页面，点击 new 新建仓库

仓库要命名为：你的github名字.github.io。注意要保持完全一致，大小写也要一样！

我的已经建好了，所以提示名字重复

其他的不用改，直接 Create repository。
